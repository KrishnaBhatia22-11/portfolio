A full-stack ML platform that predicts Air Quality Index for 26 Indian cities in real-time, explains every single prediction using SHAP game-theory explainability, and delivers personalised health advisories for 7 vulnerable population groups — children, elderly, asthma/heart patients, pregnant women, athletes. Served through a production FastAPI backend, deployed globally on Vercel. Free forever.

What no existing platform has — CPCB, SAFAR, IQAir, Breezometer — is an open, explainable, ML-driven system. This project fills that gap and is now live.

🧠
Explainable AI
Every prediction decomposed by SHAP game-theory. "PM2.5=180 → AQI +152.7 pts. CO=12.9 → AQI +82.9 pts." No other public platform does this. Recruiters see the WHY, not just the WHAT.
⚡
Live PWA + Time Machine
Progressive Web App with 26 live WAQI stations, threat matrix ranking all cities by AQI, city comparison tool (e.g. Mumbai 88% worse than Delhi), and AQI Time Machine for historical pattern analysis.
🏥
Health Intelligence
Personalised risk matrix for children, elderly, asthma/heart patients, pregnant women, athletes — across all 6 CPCB categories (Good→Severe). Precautions generated per reading. Nothing in market has this.
⚙️
Production Backend
FastAPI REST API with 3 production endpoints. Pydantic v2 validation, auto-Swagger docs at /docs. XGBoost + SHAP served as pickle artifacts. Deployed on Render.com. MLflow experiment tracking throughout.
🤖
AI Chatbot
Google Gemini 1.5 Flash integration for natural language AQI queries. "Is it safe to go running in Jaipur today?" → AI-generated personalised response using live AQI + health profile context.
📊
Manual Predictor
Any user can enter custom sensor readings (PM2.5, PM10, CO, NO2, NOx, SO2, O3, NH3) and get an instant ML prediction + SHAP explanation + health advisory. Works for cities CPCB doesn't cover.
